# Customer Churn Prediction

## Overview
This project uses a neural network (ANN) to predict customer churn and identify customers likely to leave, enabling data-driven retention strategies.

## Dataset
Customer churn dataset with demographic, account, and service usage features.

## Approach
- Data cleaning and preprocessing (encoding, normalization)  
- Feature engineering and handling categorical variables  
- Built an Artificial Neural Network for binary classification  

## Model & Training
- ANN with Dense layers and ReLU activation  
- Sigmoid output for churn prediction  
- Optimized using Adam optimizer and binary cross-entropy loss  

## Evaluation
- Accuracy, Precision, Recall, F1-score  
- Confusion Matrix and classification report  

## Results
- Achieved strong classification performance  
- Identified key factors influencing customer churn  

## Tech Stack
Python, TensorFlow, Keras, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib
